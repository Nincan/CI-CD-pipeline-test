# List Item

List is a base component to render bunch of related items.
They are composed of items containing primary and supplemental actions, which are represented by icons and text.

`autoFocus` prop makes sure that item is focused on initial mount. tabIndex must be provided to make button focusable if it is not a focusable element by default.
Can be used when items are rendered inside menu, it is more accessible if selected menu item is automatically focused when menu opens.

<Example title="List Items with button prop" path="list/examples/as-button.js">
  <AsButton />
</Example>

Hover styles are applied to the items with button prop.
If `button` is `true`, item will be focusable and onClick will be called when item is focused and enter key is pressed

<Example title="Disabled ListItem" path="list/examples/disabled.js">

</Example>

Hover styles will not be applied to disabled items.

`ListItemText` can be used when text has multiple nodes. `secondary` can be any ReactNode and is useful to display any subtext.
`AvatarListItem` is a wrapper on top of `ListItem`.

Divider with title can be used to add subheader to the list. `first` prop can be used to not add top padding and border separator.

<Example title="AlignItems usage" path="list/examples/align-items.js">

<Example title="Virtualization" path="list/examples/virtual-list.js">
  <VirtualList />
</Example>

Setting `virtual` property for the List can enable virtualization. **maxHeight and overflow must be provided to list or its container**.

Check next example if overflow cannot be handled on List component.

<Example title="Virtual List inside a container" path="list/examples/virtual-list-contained.js">
  <ContainedVirtualList />
</Example>

When virtual list is wrapped inside a container, **ref of the container must be passed to the `scrollElement` argument of the List**
