# Layout & Grid System

The Grid component is based on CSS **Grid Layout** feature, using TailwindCSS utility classes behind scenes.

The goal is to have a grid system that consist of only two components: Grid and Column (yes, no rows);
Something that was really easy to use with minimal footprint but able to build complex layouts.
