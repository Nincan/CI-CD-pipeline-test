# Switch

<Yard placeholderHeight={48} {...switchYardConfig} />

Switches are used to provide users with multiple options for selection in a series of options.

When used as a toggle they allow the user to make a binary choice usually (but not limited) in
the form of a yes/no or on/off suggestion. Toggles are often used in product settings or as filter
options. When engaged (on), Base Web toggles are colored and when disengaged (off) theyâ€™re grey.

## When to use

- When a collection of options share context.
- When a user wants to toggle an option on/off.
- When a user wants to select multiple options.
