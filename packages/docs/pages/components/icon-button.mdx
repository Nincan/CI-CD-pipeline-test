# Icon Button

<Yard placeholderHeight={54} {...iconButtonYardConfig} />

Icon buttons allow users to take actions, and make choices, with a single tap.

Refer to the <DocLink href="/iconLibrary">Icons Library</DocLink> regarding the available icon options.

For buttons with both icons and text, use the `Button` component with `leftEnhancer`. For more information, see the <DocLink href="/components/button#adding-enhancers">Button</DocLink> docs.
